# Transfer Learning con Fashion MNIST

## Descripción
Se implementa Transfer Learning en dos etapas:

### Etapa 1
Autoencoder convolucional para extracción de características.

### Etapa 2
Encoder congelado + capas densas para clasificación.

## Resultados
Accuracy en test: ~0.87

## Ramas
- master: modelo final
- etapa1: autoencoder
- etapa2: clasificador con encoder congelado
